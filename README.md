# Fees Management System (FMS)

This repository contains a Django‑based Fees Management System built as part of the IJ Education System.
It provides core modules for students, class fees, payments and autogenerated fee records, along with a
simple administrative interface and customizable templates.

## Installation

1. **Clone the repo** (see URL above).
2. Create and activate a Python virtual environment:
   ```bash
   python -m venv .venv
   # Windows
   .venv\Scripts\activate
   # macOS / Linux
   source .venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Apply database migrations and create a superuser:
   ```bash
   python manage.py migrate
   python manage.py createsuperuser
   ```
5. Start the development server:
   ```bash
   python manage.py runserver
   ```
6. Visit `http://127.0.0.1:8000/` to access the application.
